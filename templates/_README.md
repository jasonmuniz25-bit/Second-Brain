# templates

Note templates. Insert via the **Templates** ribbon icon in Obsidian (or via Templater if you want dynamic dates).

Existing templates:
- `daily-note.md` — daily note shell, used by Daily Notes plugin
- `weekly-review.md` — Sunday review shell
- `inbox-item.md` — quick capture frontmatter
- `session-summary.md` — Claude's session-end output format
- `sop.md` — Standard Operating Procedure
- `ghl-workflow.md` — GoHighLevel workflow documentation
- `airtable-base.md` — Airtable base documentation
- `roleplay-session.md` — HVAC sales call transcript + coaching
