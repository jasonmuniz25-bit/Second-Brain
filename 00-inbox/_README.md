# 00-inbox

Everything lands here first. Voice note transcripts, quick ideas, links to read later, half-formed thoughts.

When this folder gets noisy, ask Claude: *"process the inbox."* Claude reads each note, classifies it, files it into the right project folder (or archives to `AI/inbox-processed/`), and tells you what it did.

Use the `templates/inbox-item.md` template if you're capturing in Obsidian. If you're dropping in a transcript or paste, frontmatter doesn't matter — process-inbox will add it.
