# weekly-reviews

One file per week, Sunday. Filename: `YYYY-[W]ww.md` (e.g. `2026-W19.md`).

Triggered when you say *"weekly review"* on a Sunday. Claude reads the week's session summaries from `../sessions/`, plus the week's daily notes, and synthesizes:
- what got built / decided
- what got dropped or deferred
- patterns worth noticing
- top 3 priorities for next week

Use `templates/weekly-review.md`.
